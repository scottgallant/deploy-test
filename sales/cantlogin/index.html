I Can't Log In!

home > sales > cantlogin

Arg! Sorry about that. This page is here to help.

Ok, so, first we need to figure out which one of our products you are using.

Are you using the Desktop version (Balsamiq Mockups 3 for Desktop)? If so, there is no password or login required for using Mockups 3 for Desktop.  Just download, install and launch.

You also don’t need a Balsamiq login or password for:

  • Balsamiq Mockups for Confluence - log in using your Confluence account 
  • Balsamiq Mockups for JIRA - log in using your JIRA account
  • Balsamiq Mockups 3 for Google Drive - log in using your Google or Gmail account
  • Mockups To Go - you can upload mockups to the Submissions project and download from any project without logging in

The only product we offer that you do need a login and password for is myBalsamiq - the online version of Balsamiq Mockups that we also host. If you can’t remember your login or password for your myBalsamiq site, you can reset your password here.

If you’re still having trouble, please email support@balsamiq.com, preferably with the name or address of your myBalsamiq site (e.g., https://something.mybalsamiq.com/).
