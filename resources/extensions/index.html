A Thriving User Community

Adopting Balsamiq Mockups means joining a growing community of programmers, designers, product managers and usability experts who are passionate about creating software and web sites that delight users every day.

This page highlights some community tools and websites which might be useful to you. We are not affiliated with the makers of the products below unless noted.

More UI Controls

  • MockupsToGo is a user-contributed collection of ready-to-use UI components and design patterns built using Balsamiq Mockups. New patterns are added all the time - we hope you’ll want to contribute as well!
  • Mock4U is a set of UML stencils for Balsamiq Mockups.
  • Blackboard Learn Symbols is a series of assets to help design user interfaces for custom Blackboard Learn (9.0 and 9.1) extensions.
  • Charts for Mockups is a library of 18 additional Graph & Chart Symbols for Balsamiq Mockups.

Templates

  • Bootstrap and UX Deliverable Templates are included in our tutorial on creating clonable project templates.
  • Landing Pages Templates by Good UI provides a set of templates for conversion-focussed landing pages. The first sample is free.
  • Magento Wireframes is a set of templates for desktop and mobile layouts for Magento created by NuBlue.
  • Sharepoint Templates by Matthew Hughes from Flucidity.
  • Bootstrap Grid Template by Harry Budianto.

Custom Icon Packs

Icons that are ready to drop into Assets using our icon_ file naming convention:

  • Hand Drawn Icons
  • Google Material Design icons, by Jan Stawarczyk
  • Doodlekit Icons
  • Font Awesome Icons (Font Awesome is now included with Balsamiq Mockups 3!)

Instructions on how to use custom icon packs are here.

Other Products That Work with Mockups (Versions 2 and/or 3)

Balsamiq Mockups version 3 uses a new file format that is not compatible with tools that use BMML or XML formats. If you want to use them you should read about importing or exporting BMML files.

Note that we are not affiliated with any of these tools and can’t ensure they will work as desired or continue to be supported.

  • Frontify, a web app for creating specifications, light prototypes, and style guides from your Mockups PNG files and myBalsamiq prototypes.
  • Flairbuilder, a prototyping tool that imports BMML
  • iMockups is an iPad application that exports to BMML. An iPad application is on our roadmap too.
  • If you work with SWT, you’ll like this open-source SWT to BMML importer
  • LiveMockups is an application that helps you show your Mockups prototypes on an iPhone or iPad
  • Wirify Pro is a bookmarklet which turns web pages into BMML mockups
  • Mockups.me is an iPad, Android, Web and Desktop wireframing app that imports and exports BMML files
  • mockupsNode is a NodeJS module that reads, manipulates, and writes BMML files
  • Mockups Strings Extractor is a Python script to extract text strings out of Balsamiq Mockups files (Open Source!)
  • Advanced Mockup Strings Extractor is a script to extract Mockups text into XML readable by translation memory systems
  • Excel Calendar Tool for Balsamiq is an Excel file that allows you to easily generate Balsamiq Mockups markup for a calendar showing a month (or two) that you specify
  • Table Editor is an online tool for entering tabular data that you can export to Balsamiq Mockups-formatted text
  • ReqWriter Pro generates product requirement documents from Balsamiq Mockups files
  • SpeedWapp imports BMML files to their dynamic web page builder
  • Mobile Phone Gestures is a Balsamiq file that contains images of hands performing gestures to overlay on the iPhone control.
  • Indigo Studio can import BMML files so that you can turn them into interactive prototypes.
  • myBalsamiq Filters is a Google Chrome extension to filter your myBalsamiq projects by name or cover style.

Exporting Your Mockups to Code (Version 2)

Balsamiq Mockups version 3 uses a new file format that is not compatible with most of these tools. If you want to use them you should read about importing or exporting BMML files.

Note that we are not affiliated with any of these tools and can’t ensure they will work as desired or continue to be supported.

Web

  • to HTML/CSS: nedeco converter (freeware)
  • to HTML/JS or GWT: Reify transforms mockups into SmartClient or Smart GWT applications (commercial, blog post intro)
  • to HTML/CSS/JS: try Napkee (blog post intro)

Flash

  • to MXML/AS: try Napkee (blog post intro)
  • to MXML: try QXmlEdit (open-source)

Other Applications

  • to a 3-tier application: try the WebORB Exporter
  • to a live application using MS Word: try SoftwareServant (blog post intro)
  • to XAML: learn about the MockupToXaml project (open-source)

Want to run Mockups via the command-line on a headless server? Ian Norton created these open-source scripts to help.
