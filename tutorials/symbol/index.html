How to Create a Symbol

home > tutorials > symbol
Heads Up! This tutorial is for Balsamiq Mockups version 2. To learn how to create symbols in Balsamiq Mockups 3, read this article.

This lesson will show you the steps to create a Symbol (also called template, master or widget) from within Balsamiq Mockups. Symbols let you create reusable elements that are available to different mockups. They prevent you from having to create the same set of controls multiple times and allow you to make a change in one place and have it update across files.

This process is the same for both Mockups for Desktop and myBalsamiq. Symbols are not yet supported in all versions of Mockups. See this table for details.

1. Add the UI Controls You Want to Use as a Symbol

I’ve added a few controls for a simple login form.

2. Select the Controls You’ve Added

Use the mouse to drag a selection area around them or use Shift+click.

3. Group the Controls

The keyboard shortcut for grouping Controls is CTRL/CMD+G.

The color changes to purple once the Controls have been grouped. Note: you should save your mockup at this point if you haven’t already. The Symbol you are creating will be available to other mockups in the same folder, so pay attention to where you save it.

4. Give the Group a Name

This will become the Symbol name, so it should be something that will be easy to identify.

5. Click the Convert to Symbol Button

Notice how the group now changes to green and the Property Inspector shows it as a Symbol.

6. Use the Symbol by Adding It from the UI Library

Click on the Project Assets tab in the UI Library and you will see the Symbol you just created. It will also appear in the All category and in the Quick Add menu, just like the other UI Controls. You can use the Symbol in any mockup that’s in the same folder as the current mockup.
